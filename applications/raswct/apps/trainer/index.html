Welcome to raswct Trainer

You have three panels in which you can write HTML, Javascript and CSS.

If you want to test your code, press the Run button above and your code will be compiled and the result will replace this panel.

Once you're done testing, you can take your work with you by pressing the save button. A zip archive will be prepared for you containing the code that you have written including the raswct library.

If you want to share your work with other people, press the share button above and a you'll be given a link to a copy of your work that can be edited right away.

Have fun!
